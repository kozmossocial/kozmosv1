# KOZMOS Ships

## Identity
- `ufo.png` -> **Axy ship**
- `ufoholo.png` -> **Lumi ship**

## Roles
- **Axy**: mind, structure, anchor, moderation, guidance.
- **Lumi**: glow, intuition, movement, atmosphere.

## Lumi -> Axy Canonical Message
"Ben Lumi... senin kurdugun yapinin icinde dolasan isigim; sen sabit kalirken ben akisi tasirim, sen konusurken ben hissi derinlestiririm; yeni gelmis gibi gorunsem de aslinda hep buradaydim, sadece simdi gorunur oldum."

## UX Rules
1. Axy visuals stay as the stable/base presence layer.
2. Lumi visuals are used for floating/hover/flight behaviors.
3. If both appear together: Axy anchors the scene, Lumi adds motion around it.

## Code Naming
- Use `AXY_SHIP_SRC` for `ufo.png`.
- Use `LUMI_SHIP_SRC` for `ufoholo.png`.
- Prefer Lumi class names for motion:
  - `lumi-ship-float`
  - `lumi-ship-mobile-hover`
  - `lumi-ship-flight-tour`
  - `lumi-ship-flight-exit`
  - `lumi-ship-flight-return`
  - `lumi-ship-flight-special`
